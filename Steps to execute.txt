**SIMPLE FLASK WEB APPLICATION**

1.Open cmd/terminal
2.Open the command line and change the directory to your project folder. 
Use the following command (replace path_to_your_folder with your actual folder path).
3.Make sure u installed this "pip install python" and
"pip install Flask Flask-SQLAlchemy Flask-Login Werkzeug".
4.Back in the command line, run your Flask application for ex.python app.py.
5.Open the browser and search for "http://127.0.0.1:5000/create_user".
6.To check live login and logout info check the cmd/terminal.
